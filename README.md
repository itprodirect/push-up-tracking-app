# Push-Up Tracking App

Small Vite + React workout tracker with local `localStorage` persistence for push-up logging and workout logging.

## Run Locally

```bash
npm install
npm run dev
```

## Deployment

The app is currently deployed as a standard web app build from Vite + React.
Offline/installable PWA behavior is not part of the current deployment.

## Validation

```bash
npm test
npm run build
```
